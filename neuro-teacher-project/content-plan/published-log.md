## Шаблон строки для новой записи

```
| YYYY-MM-DD | [краткая тема] | ВК или ТГ | охват: __, лайки: __, комменты: __, пересылки: __ |
```

## Что отслеживать

| Метрика | ВК | ТГ |
|---|---|---|
| Основная | Охват | Просмотры |
| Вовлечение | Лайки + комменты + пересылки | Реакции + пересылки |
| Глубина | Дочитывания (если есть) | Время чтения |
| Конверсия | Переходы по ссылкам | Переходы по ссылкам |

## Как пользоваться

1. Сразу после публикации — скопируй строку из `content-plan-month.md` сюда, статус смени на ✅.
2. Через 24 часа — заполни статистику (охват/лайки/комменты).
3. Через 7 дней — перенеси итоговые цифры в `analytics-log.md` с выводами.
