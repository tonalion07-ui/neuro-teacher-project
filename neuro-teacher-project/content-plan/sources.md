## Как искать новые темы

1. **Раз в месяц** — пройтись по ТОП-5, выписать 5–10 идей в `topics-rotation.md`.
2. **Раз в неделю** — заглянуть в новостные источники (Минпросвещения, EdTech-каналы).
3. **После каждой встречи/созвона с учителем** — записать в `neuro-teacher-project/` цитату и тему.
4. **Раз в месяц** — спросить аудиторию (опрос в ВК/ТГ), какую тему раскрыть следующей.

## Правила цитирования

- Цитата из внешнего источника → дать ссылку.
- Цитата учителя → дать имя (с разрешения) или обезличить.
- Статистика → источник + дата.
- Скриншот чужого поста → только с указанием автора.
